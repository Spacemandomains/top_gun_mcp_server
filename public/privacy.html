Privacy Policy

Last updated: May 2026

TOP GUN GEO-Lens is a pay-per-query API service. We collect only what is necessary to process your request and complete payment. We do not sell your data.

1. What We Collect

When you use GEO-Lens we collect:

  • Query strings — the brand name or keyword you submit for auditing.
  • Payment tokens — the Stripe session ID or x402 payment proof you provide. We do not store full card details.
  • Server logs — standard HTTP request metadata retained for up to 30 days.

2. What We Do Not Collect

  • We do not require account registration or store user profiles.
  • We do not use tracking pixels, advertising cookies, or third-party analytics.
  • We do not retain audit results after the API response is returned.

3. How We Use Your Data

Data collected is used solely to execute the search and scoring pipeline, verify payment, and maintain service security.

4. Third-Party Services

  • Stripe — payment processing. Governed by Stripe's Privacy Policy.
  • Brave Search API — search index queries.
  • Exa API — semantic search index queries.
  • Vercel — hosting. Governed by Vercel's Privacy Policy.

5. Data Retention

Query strings and payment tokens are not persisted to a database. Server access logs are retained for up to 30 days, then deleted.

6. Security

All traffic is served over HTTPS. Payment verification is performed server-side. Secret keys are never exposed to the client.

7. Your Rights

Because we do not maintain user accounts or persistent query records, there is no personal profile to access or delete. Contact us via the repository below with any questions.

8. Children's Privacy

GEO-Lens is a developer API service not directed at children under 13.

9. Changes to This Policy

Material changes will be reflected in the "Last updated" date above.
